## Deploy (GitHub Pages)
1) อัปโหลดไฟล์ทั้งหมดขึ้น repo
2) Settings → Pages → Deploy from a branch → Branch: main, Folder: /(root) → Save
3) เปิด URL ที่ GitHub ให้มา

ข้อมูลคำอธิษฐานเก็บไว้ใน LocalStorage ฝั่งผู้ใช้เท่านั้น
