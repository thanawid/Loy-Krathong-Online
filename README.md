## Deploy (GitHub Pages)
1) อัปโหลดไฟล์ทั้งหมดขึ้น repo
2) Settings → Pages → Deploy from a branch → Branch: main, Folder: /(root) → Save
3) เปิด URL ที่ GitHub แสดง

ข้อมูลคำอธิษฐานเก็บไว้ที่ LocalStorage ของผู้ใช้เท่านั้น
